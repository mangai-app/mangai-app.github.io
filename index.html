Welcome to MangAI

MangAI uses advanced AI technology to colorize all your favorite black and white manga!

How to Use

  • Download the extension on the chrome web store
  • Navigate to one of our supported websites
  • Turn on the colorize feature within the extension
  • Read away!

The app is currently running in closed alpha. Changes will be made as we learn from our users. If you would like access to the closed alpha, please join our public discord.

Supported websites

MangAI will attempt to run on all sites while it is toggled on. The following sites have been tested and given additional support:

  • mangakakalot.com
  • mangago.me
  • bato.to
  • mangapark.net
  • kissmanga.nl
